Pumpa

Pumpa is a generic white-label solution for both iOS and Android for a loyalty program using beacons technology, integrated with Practi system. Already deployed for more than 10 end customers.

Bright Inventions provides the whole development services, including backend and on-site deployment support. The apps consisted of push notifications, Google Maps and Facebook integration and dynamic UI with both left-to-right and right-to-left writing systems support.
